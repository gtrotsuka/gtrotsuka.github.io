ABOUT / SYSTEM PROFILE

A software profile shaped by robotics, interfaces, and operating real systems.

I’m a CS master’s student at Georgia Tech interested in robotics, interactive systems, and software that makes complicated systems easier to use. I like building tools that behave predictably, explain themselves clearly, and still feel good to work with.

That usually means sitting somewhere between low-level engineering and the people using the system: C++ and Qt applications, autonomy tooling, data-processing workflows, web interfaces, and teaching infrastructure that helps students get unstuck.
